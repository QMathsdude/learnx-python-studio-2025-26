# ✨ Lesson 2 : Git & Github


Welcome to the first session of our Python workshop! 🐍   
In this lesson, we will **set up version control software** that will enable you to *create and maintain your own online repository*. ⚙️

By the end of this session, you should: 🚀
- Understand what is Git and it's functionality
- Use local Git to manage Python projects
- Understand what is Github and create your own account
- Backup Python project onto a Github repository

Please take out a laptop and **follow along the video tutorials! 💻**

## ⚙️ Lesson 2 : Git & Github

[![Lesson-2](../images/Lesson-2.png)](https://youtu.be/RnWDKWmaQ8s?si=2dc5dsKY4GjW1ZoR)
[Watch the video here](https://youtu.be/RnWDKWmaQ8s?si=2dc5dsKY4GjW1ZoR)

### Summary

This lesson introduces you to what Git is and also how it works under-the-hood. Next, teaches you basic Git commands in order to version control your projects. Then, introduces you to what Github and also how to link your local repository to online Github repository.

### Tasks
For this lesson, **please follow along**. By the end of the video, you should have :
- Installed Git on your computer.
- Connected `conda` to Git Bash. 
- Created `alias` in Git Bash to shorten lengthy commands.
- Linked local repository (Git) to online repository (Github).
- `git clone` this repository, and pasted `Week_01` and `Week_02` into your own private repository before pushing it to Github.
